Panduan Penggunaan Aplikasi Kasir Suka Maju Bersama
Menu Utama
Aplikasi ini memiliki 10 fitur utama yang dapat diakses melalui menu utama:

1. Tambah Barang
2. Edit Barang
3. Hapus Barang
4. Tampilkan Semua Barang
5. Tampilkan Metode Pembayaran
6. Proses Transaksi
7. Buat Laporan Penjualan
8. Simpan Transaksi ke File
9. Tampil Semua Laporan Transaksi
0. Keluar

Cara Penggunaan Setiap Fitur
1. Tambah Barang

Pilih menu "1" pada menu utama
Masukkan ID Barang (harus unik)
Masukkan Nama Barang (harus unik)
Masukkan Harga Barang (tidak boleh negatif)
Masukkan Stok Barang (tidak boleh negatif)
Sistem akan menanyakan apakah ingin menambah barang lagi (y/n)
Pilih 'y' untuk menambah barang lain atau 'n' untuk kembali ke menu utama

2. Edit Barang

Pilih menu "2" pada menu utama
Masukkan ID Barang yang ingin diubah
Jika barang ditemukan, masukkan:

Nama Barang baru
Harga Barang baru
Stok Barang baru


Sistem akan menyimpan perubahan dan kembali ke menu utama

3. Hapus Barang

Pilih menu "3" pada menu utama
Masukkan ID Barang yang ingin dihapus
Sistem akan menghapus barang jika ditemukan
Tekan tombol apa saja untuk kembali ke menu utama

4. Tampilkan Semua Barang

Pilih menu "4" pada menu utama
Sistem akan menampilkan tabel berisi semua barang dengan informasi:

ID Barang
Nama Barang
Harga Barang
Stok Barang


Tekan tombol apa saja untuk kembali ke menu utama

5. Tampilkan Metode Pembayaran

Pilih menu "5" pada menu utama
Sistem akan menampilkan metode pembayaran yang tersedia:

Tunai
Kartu Kredit
Dompet Digital


Tekan tombol apa saja untuk kembali ke menu utama

6. Proses Transaksi

Pilih menu "6" pada menu utama
Masukkan nama barang yang ingin dibeli
Masukkan jumlah barang yang diinginkan
Ulangi langkah 2-3 untuk barang lain
Ketik 'sudah' jika selesai memilih barang
Pilih metode pembayaran (1-3)
Sistem akan menampilkan struk belanja dengan rincian:

Daftar barang yang dibeli
Jumlah masing-masing barang
Harga per barang
Subtotal per barang
Total keseluruhan


Tekan tombol apa saja untuk kembali ke menu utama

7. Buat Laporan Penjualan

Pilih menu "7" pada menu utama
Sistem akan menampilkan tabel berisi semua transaksi dengan informasi:

Nomor transaksi
Nama barang
Jumlah
Total harga
Metode pembayaran
Waktu transaksi


Tekan tombol apa saja untuk kembali ke menu utama

8. Simpan Transaksi ke File

Pilih menu "8" pada menu utama
Sistem akan menyimpan semua transaksi ke dalam file
Tekan tombol apa saja untuk kembali ke menu utama

9. Tampil Semua Laporan Transaksi

Pilih menu "9" pada menu utama
Pilih filter laporan:

Laporan Harian (masukkan tanggal, bulan, tahun)
Laporan Bulanan (masukkan bulan, tahun)
Laporan Tahunan (masukkan tahun)


Sistem akan menampilkan transaksi sesuai filter yang dipilih
Tekan tombol apa saja untuk kembali ke menu utama

Cara Keluar dari Aplikasi

Pilih menu "0" pada menu utama untuk keluar dari aplikasi
